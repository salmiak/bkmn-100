# bkmn-100

A minimalistic static website showcasing an AI conversation interface with a black background and clean design.

## Features

- **Minimalistic Design**: Black background with subtle borders and clean typography
- **Conversation Bubbles**: User messages appear on the right, AI responses on the left
- **AI-Generated Images**: Images are embedded within the conversation flow
- **Responsive**: Works seamlessly on desktop and mobile devices
- **No Dependencies**: Pure HTML, CSS, and SVG - no external libraries needed

## How to View

Simply open `index.html` in your web browser, or serve it with any static web server:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js (with http-server)
npx http-server

# Using PHP
php -S localhost:8000
```

Then navigate to `http://localhost:8000` in your browser.

## Files

- `index.html` - Main HTML structure with conversation content
- `style.css` - Styling for the minimalistic black theme
- `image1.svg` - Futuristic city illustration
